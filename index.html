What is open sourcing?

As we all know that there are thousands of projects and softwares whose code is available to all and anyone can make changes in it and also report bugs in the software/project and such softwares/projects are called Open source projects.

Anyone who contributes to such open source projects are called Open source contributors. Now that doesn't mean that you can contribute through only coding, you can also improve the documentation of the project, it is also an open source contribution.
